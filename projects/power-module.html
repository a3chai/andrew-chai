Overview

The High Current Power Module is a power distribution board designed for applications requiring high current delivery with accurate monitoring capabilities. It provides multiple regulated outputs with integrated current and voltage sensing.

The module is designed for drone and robotics applications where reliable power distribution and battery monitoring are critical for safe operation.

Key Features

  • High current capacity up to 120A continuous
  • Integrated voltage and current monitoring
  • Multiple regulated 5V and 12V outputs
  • Battery voltage sensing with cell balancing connector
  • Overcurrent and reverse polarity protection
  • Low voltage cutoff protection
  • I2C interface for telemetry data
  • Compact mounting pattern for drone frames

Technical Specifications

Input Voltage 3S-8S LiPo (11.1V - 33.6V)
Max Current 120A continuous, 150A burst
5V Output 5V @ 3A (switching regulator)
12V Output 12V @ 2A (switching regulator)
Current Sensing Hall-effect sensor, 0.1A resolution
Voltage Sensing Per-cell monitoring, 0.01V resolution
Interface I2C, Analog output
Board Size 50mm x 50mm

Design Details

Power Path

The main power path uses heavy copper pours (2oz outer, 1oz inner) with multiple parallel paths to minimize resistance. High-current XT60/XT90 connectors are used for battery and ESC connections.

Current Sensing

A bidirectional Hall-effect current sensor provides isolated current measurement without introducing series resistance. The sensor output is conditioned and available on both the I2C bus and an analog output pin.

Voltage Regulation

Synchronous buck converters provide efficient 5V and 12V rails for flight controllers, cameras, and other peripherals. Both outputs feature soft-start and overcurrent protection.
