Overview

The Strain Gauge Data Logger is a precision multi-channel data acquisition system designed for capturing strain gauge measurements in structural testing applications. It features 8 independent channels, each with a dedicated 24-bit sigma-delta ADC, programmable gain instrumentation amplifier, and Wheatstone bridge excitation.

Data is streamed over USB to a host PC for real-time visualization and logging. The system supports sampling rates up to 1kSPS per channel with simultaneous acquisition across all channels.

Key Features

  • 8 independent measurement channels
  • 24-bit sigma-delta ADCs (ADS1256 or similar)
  • Programmable gain from 1x to 128x
  • Precision voltage reference with <5ppm/°C drift
  • Isolated Wheatstone bridge excitation (2.5V or 5V)
  • Quarter, half, and full bridge support
  • USB 2.0 FS interface with CDC class
  • Optional SD card logging for standalone operation
  • STM32F4 microcontroller for data processing

Technical Specifications

Channels 8 differential inputs
Resolution 24-bit (effective ~20-bit at max gain)
Sample Rate Up to 1kSPS per channel (simultaneous)
Input Range ±10mV to ±5V (programmable)
Excitation 2.5V or 5V, 50mA max per channel
Interface USB 2.0 Full Speed (12Mbps)
Power USB powered or 9-12V DC input
Board Size 100mm x 80mm

Design Details

Analog Frontend

Each channel features a precision instrumentation amplifier (INA333) followed by a dedicated ADC. The bridge excitation is provided by a precision current source to minimize errors from lead resistance. Anti-aliasing is handled by a 2nd-order Butterworth filter before the ADC.

Grounding and Isolation

The analog and digital sections are separated with a split ground plane, joined at a single point under the ADC. Bridge excitation can optionally be isolated using onboard DC-DC converters and digital isolators for applications requiring galvanic isolation.

Calibration

The system includes onboard calibration routines for offset and gain correction. A precision shunt calibration resistor can be switched in for automated sensitivity calibration of each channel.
